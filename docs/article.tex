Введение.
Береговой линий называется место пересечения суши и водных объектов. Граница на изображении описывается контуром состоящим из последовательности точек. Для того чтобы береговая линия была достаточно точной, часто ее отрисовывают вручную. Для этого используют снимки с разных спутников и разных сезонов, так как на некоторых снимках может быть лед на воде, весной паводок, а осенью межень. Но ручная разметка занимает много времени, а в открытом доступе нет инструмента???? для автоматической разметки береговой линии обладающего достаточной точность
Заказчик проекта размечает карты в QGIS. Это свободная геоинформационная система. Данная геосистема предназначена для создания и редактирования данных, производства карт и выполнения аналитических операций.
Образ результата.
Требовалось разработать алгоритм автоматической разметки береговой линии и реализовать с помощью плагина для QGIS в котором при выделении область на карте будет выведены все береговые линии в этой области с привязкой к координатам.
